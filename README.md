# HELLO WORLD
 > A simple python GUI interface made experimenting with Tkinter.

Uma aplicação simples em **Python + Tkinter** criada para praticar conceitos de **GUI**, organização do código em módulos e manipulação de imagens com **Pillow (PIL)**.

A interface começa com uma tela inicial em formato de “popup” e, ao clicar, troca para uma cena com elementos visuais ("Hello World")

## Funcionalidades

- Tela inicial 
- Troca de tela ao iniciar
- Renderização de imagens e elementos visuais na janela
- Animação simples de “pulo” para as letras (efeito visual)
- Código separado em módulos (`main.py`, `janela.py`, `popup.py`, `action.py`)

## Tecnologias

- **Python**
- **Tkinter**
- **Pillow (PIL)** para abrir/redimensionar imagens

## Como executar

#### Pré-requisitos
- Python instalado
- Pillow instalado

#### Instalação
```bash
pip install pillow
```

#### Rodar
```bash
python main.py
```
